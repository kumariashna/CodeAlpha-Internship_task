# 📊 CodeAlpha Internship Project

This repository contains Power BI dashboards developed during the CodeAlpha Power BI Internship. Each project focuses on solving real-world business problems through data analysis and interactive visualizations.

---

# 📊 Task 1: Financial Health Dashboard

### Description:

An interactive Power BI dashboard designed to analyze an organization's financial performance through revenue, profit, expenses, balance sheet, and cash flow analysis. It helps stakeholders monitor financial health and make data-driven decisions.

### Features:

- Revenue & Expense Analysis
- Profit & Loss Overview
- Balance Sheet Analysis
- Cash Flow Analysis
- Financial KPI Cards
- Trend Analysis
- Interactive Filters & Slicers

### Tools Used:

- Power BI
- Power Query
- DAX
- Excel/CSV
